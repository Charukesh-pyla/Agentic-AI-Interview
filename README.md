# Agentic AI–Driven Interview Question Generator

## Problem Statement
Traditional interview question selection is manual, generic, and inefficient. 
This project builds a fully agentic AI system that autonomously plans, reasons, 
and adapts to generate personalized interview questions.

## Why Agentic AI?
- Autonomous agents
- Goal-oriented planning
- Reasoning & explainability
- Human-in-the-loop control
- Memory & adaptability

## Tech Stack
- Python
- LangGraph
- LangChain
- LLMs
